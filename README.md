## Deskripsi
Project ini merupakan demo praktikum **Testing API** pada mata kuliah **Web Service Engineering (WSE)**.  
Aplikasi dikembangkan menggunakan **Node.js** dan **Express.js**, serta memanfaatkan **API publik JSONPlaceholder** sebagai data dummy.  
Pengujian endpoint API dilakukan menggunakan **Postman** dengan metode HTTP **GET, POST, PUT, dan DELETE**.

---

## Tujuan Praktikum
1. Memahami konsep dasar Web Service dan REST API  
2. Mengenal dan menggunakan metode HTTP (GET, POST, PUT, DELETE)  
3. Melakukan pengujian API menggunakan Postman  
4. Memahami pertukaran data berbasis JSON  

---

## Teknologi yang Digunakan
- Node.js  
- Express.js  
- Axios  
- Postman  
- JSONPlaceholder API
